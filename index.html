Langkah Pertama

Aktivasi Lisensi & Login Pengguna

Saat pertama kali dijalankan, aplikasi akan meminta kunci lisensi guna memverifikasi legalitas pemasangan program di komputer Anda.

  • 1
    Salin Hardware ID (HWID) Anda dari layar aktivasi awal aplikasi. HWID dibentuk unik berdasarkan CPU dan Hardware komputer.
  • 2
    Kirim HWID tersebut ke WhatsApp Admin untuk mendapatkan Kunci Lisensi yang sah.
  • 3
    Masukkan Kunci Lisensi ke input field lalu tekan tombol Aktifkan Sekarang. Program otomatis terbuka secara penuh.
  • 4
    Pada layar login berikutnya, gunakan kredensial default untuk masuk: admin (password: admin) atau kasir (password: kasir).

Fungsi Utama

Operasional Modul POS (Point of Sale)

Modul kasir dirancang dengan mengutamakan kecepatan entri data. Kasir dapat melakukan seluruh transaksi murni menggunakan keyboard.

  • 1
    Tekan tombol F2 untuk meletakkan kursor langsung ke kolom pencarian barcode.
  • 2
    Scan barcode produk dengan pemindai laser, barang akan langsung ditambahkan ke keranjang belanja sebelah kanan dengan kuantitas 1.
  • 3
    Jika barang tidak memiliki barcode, klik input pencarian nama produk di sebelahnya atau cari manual dengan memfilter kategori.
  • 4
    Tekan tombol F4 untuk membuka jendela pembayaran cepat. Masukkan nominal uang tunai yang diterima, pilih jenis pembayaran (Tunai, Debit, QRIS), lalu tekan Enter untuk cetak struk.

Pengelolaan Toko

Manajemen Master Produk & Stok

Melalui menu Produk, administrator dapat memantau seluruh ketersediaan barang di minimarket secara komprehensif.

  • 1
    Klik tab menu Produk (Stok) di sidebar kiri aplikasi (Hanya dapat diakses oleh user berkedudukan Admin).
  • 2
    Gunakan filter Tampilkan Stok Kritis saja untuk melihat produk mana saja yang stoknya di bawah batas minimal kritis.
  • 3
    Klik Tambah Produk Baru untuk membuka form input data baru (nama, harga beli, harga jual, barcode, serta kategori).
  • 4
    Gunakan fitur Ekspor Excel (.xlsx) untuk mengunduh seluruh data barang ke format tabel Excel dalam satu detik.

Keamanan Data

Sistem Cadangan (Backup) & Sinkronisasi

Untuk menjaga keberlanjutan bisnis minimarket, aplikasi menyediakan fitur pencadangan data ganda baik secara lokal maupun cloud.

  • 1
    Masuk ke menu Pengaturan Toko, cari bagian panel Pemeliharaan & Database di bagian bawah.
  • 2
    Backup File Lokal: Klik tombol Ekspor File JSON (Backup) untuk mengunduh seluruh database aplikasi ke file cadangan JSON lokal.
  • 3
    Backup Firebase Cloud: Pastikan komputer terhubung dengan internet, lalu klik Cadangkan ke Cloud (Firebase) untuk mengunggah database secara instan ke server cloud Anda.
  • 4
    Restore Data: Jika komputer kasir bermasalah atau diganti, Anda cukup mengimpor file backup JSON lokal atau menekan tombol Pulihkan dari Cloud untuk mengembalikan seluruh kondisi data seperti semula.
